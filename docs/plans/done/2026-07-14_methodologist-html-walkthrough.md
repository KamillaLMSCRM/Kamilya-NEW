# Интерактивная HTML-демонстрация флоу методолога

## Цель

Создать самостоятельную HTML-презентацию Kamilya LMS, которая последовательно показывает реальный продуктовый путь: внутренний документ компании → подготовленный курс и тест → назначение → прохождение → сертификат и журнал обучения. Презентация должна работать без API и подходить для показа клиенту и записи видеоролика.

## Пункт 1 — собрать проверенный сценарий и визуальные ориентиры

- Сверить роли, маршруты и состояния с текущим frontend и production E2E.
- Использовать только подтвержденные функции; явно не заявлять юридическую значимость журнала или сертификата.
- Выбрать нейтральный демонстрационный документ и компанию без персональных данных.

**Что сделано:** сценарий сверён с текущими маршрутами frontend, production AI smoke и first-tenant E2E. В walkthrough используются подтверждённые состояния: загрузка документа, AI-черновик, редактура, тест, публикация, назначение, прохождение, сертификат и журнал.

**Проверки:** `docs/reports/2026-07-14_ai-production-smoke.md`, `docs/reports/2026-07-14_first-tenant-e2e.md`, текущий `Sidebar.tsx` и страницы соответствующих маршрутов.

**Статус:** ✅ done
