# Презентации Kamilya LMS

## Интерактивный walkthrough методолога

Файл: `2026-07-14_methodologist-product-walkthrough.html`

Презентация работает локально без внешних библиотек и API. Для предсказуемой навигации по query-параметрам откройте её через локальный HTTP-сервер или как обычный HTML-файл в Chromium/Chrome/Edge.

### Режимы

- `mode=sales` — короткий клиентский сценарий из 8 экранов; используется по умолчанию;
- `mode=full` — подробный продуктовый walkthrough из 15 экранов.

Примеры:

```text
2026-07-14_methodologist-product-walkthrough.html?mode=sales
2026-07-14_methodologist-product-walkthrough.html?mode=full
2026-07-14_methodologist-product-walkthrough.html?mode=sales&slide=3
```

### Управление

- `←` / `→` — предыдущий и следующий экран;
- `Space` — запустить или остановить автопоказ;
- `O` — обзор всех экранов;
- `C` — скрыть элементы управления;
- `M` — переключить короткий и полный режимы;
- `Home` / `End` — первый и последний экран;
- свайп — навигация на сенсорном устройстве.

Текущий режим и номер экрана сохраняются в URL. Ссылку можно отправить сразу на нужный кадр.

### Режим записи ролика

Откройте файл с query-параметрами:

```text
2026-07-14_methodologist-product-walkthrough.html?mode=sales&record=1
```

Режим `record=1` включает автопоказ, скрывает управляющие элементы и курсор. Для ручного запуска без элементов управления используйте `?clean=1`, для запуска с конкретного экрана — `?slide=5`.

Рекомендуемый формат записи: Chromium, viewport `1920×1080`, без панели закладок и уведомлений. Короткая версия рассчитана примерно на 1 минуту, полная — примерно на 1,5 минуты без голосовой дорожки. Для ролика с озвучкой длительности экранов следует синхронизировать с финальным текстом диктора.
