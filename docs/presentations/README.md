# Презентации Kamilya LMS

## Интерактивный walkthrough методолога

Файл: `2026-07-14_methodologist-product-walkthrough.html`

Презентация работает локально без сервера, внешних библиотек и API. Откройте HTML в Chromium/Chrome/Edge.

### Управление

- `←` / `→` — предыдущий и следующий экран;
- `Space` — запустить или остановить автопоказ;
- `O` — обзор всех экранов;
- `C` — скрыть элементы управления;
- `Home` / `End` — первый и последний экран;
- свайп — навигация на сенсорном устройстве.

### Режим записи ролика

Откройте файл с query-параметрами:

```text
2026-07-14_methodologist-product-walkthrough.html?record=1
```

Режим `record=1` включает автопоказ и скрывает управляющие элементы. Для ручного запуска без элементов управления используйте `?clean=1`, для запуска с конкретного экрана — `?slide=5`.

Рекомендуемый формат записи: Chromium, viewport `1920×1080`, без панели закладок и уведомлений. Полная версия рассчитана примерно на 1,5 минуты без голосовой дорожки. Для ролика с озвучкой длительности экранов следует синхронизировать с финальным текстом диктора.
